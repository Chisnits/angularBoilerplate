#AngularJS boilerplate 

Includes:
* ui-router
* sass
* gulp

Fork or clone, then run npm install.


##Folder structure:

###dist
- all.css
- all.js

###js
- controllers
    - mainCtrl.js
- directives
    - mainDirective.js
    - navbar.js
- services
    - mainService.js
- app.js

###styles
- base
    - reset.css
- fonts
    - fonts.css
- views
    - main.css
    - main.scss

###views
- home.html
- navbar.html
- random.html

###root level 
- gulpfile.js
- index.html
- package.json
- readme.md
